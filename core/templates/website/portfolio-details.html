Regal Mehrad – Full‑Stack E‑commerce Platform

regalmehrad.example.com

A fully functional online store with product catalog, shopping cart, user authentication, order management, and secure payment integration – built from scratch with Django.

Regal Mehrad is an e‑commerce website for a fashion brand. The client needed a custom online store that is easy to manage, mobile‑friendly, and scalable. I developed the entire application using Django for the backend, PostgreSQL for the database, and Bootstrap for a responsive frontend. Key features include product filtering, user profiles, order history, and an admin dashboard for inventory management.

The main challenge was building a secure and smooth checkout process while keeping the admin interface intuitive for non‑technical staff. Additionally, the site needed to handle product variants (size, color) and integrate a third‑party payment gateway (PayPal) without compromising performance.

I built a modular Django application with a custom product model supporting variants, implemented a session‑based cart that persists after login, and integrated PayPal REST API for payments. The admin panel was extended to allow easy product and order management. The frontend uses Bootstrap 5 with AJAX for cart updates without page reloads.

Key Features

  • Product catalog with search & filters
  • User registration & profiles
  • Shopping cart & wishlist
  • Secure checkout (PayPal)
  • Order tracking & email notifications
  • Admin dashboard for inventory
